Chrome版网申助手安装方法

亲爱的小伙伴, 由于伟大防火墙的原因, 导致Chrome网上应用店无法访问, 能翻墙的小伙伴, 可以直接去Chrome网上应用店安装.
如果不能翻墙也不要紧, 只需三步即可轻松解决安装问题:

步骤一
下载安装包：鼠标右键点击下方链接，选择“链接另存为”，将助手安装包保存到本地： http://shared.ydstatic.com/gouwuex/chrome/4.2.4/youdaogouwu-4.2.4.crx
步骤二
打开扩展管理页面：点击浏览器右上角的设置图标—>选中“工具”或“更多工具”—>选中“扩展程序”：
步骤三
将刚才下载的惠惠购物助手安装包拖到扩展管理页面中完成安装。
